# Enable Agentforce for Guided Shopping

Turn on Agentforce for Guided Shopping for your B2C storefront.

1.  In your Salesforce org, at the top of the page, click ![Setup](https://sf-zdocs-cdn-prod.zoominsoftware.com/tdta-b2c_merchandiser_administrator-2-production-enus/beac8152-03cc-40b1-b594-e8ed108b8b62/images/setup_icon_no_border.png) and select **Setup**.
2.  In the Quick Find box, enter and select **Commerce Agentforce Settings**.
3.  Turn on **Agentforce for Guided Shopping â€” B2C**, then refresh your browser.

    ![Commerce Agentforce Settings in setup](https://sf-zdocs-cdn-prod.zoominsoftware.com/tdta-b2c_merchandiser_administrator-2-production-enus/beac8152-03cc-40b1-b594-e8ed108b8b62/images/b2c_commerce_agentforce_settings.png)

4.  In the Quick Find box, enter and select **Agentforce Agents**.

    If you don't see this option, check if Einstein Generative AI is turned on. Make sure that you have the correct license and required user permissions for your agent type.

5.  Turn on Agentforce, and then refresh the page.
6.  Confirm that you can see the **New Agent** button.

    ![Agentforce Agents setting](https://sf-zdocs-cdn-prod.zoominsoftware.com/tdta-b2c_merchandiser_administrator-2-production-enus/beac8152-03cc-40b1-b594-e8ed108b8b62/images/b2c_agentforce_agents_setting.png)

    If you don't see the button, make sure that you have permissions for at least one agent type other than Agentforce (Default).
